# 🔐 Secure Authentication Module for Operating Systems

## 📌 Overview
The **Secure Authentication Module for Operating Systems** is a web-based simulation project that demonstrates how modern operating systems implement advanced security mechanisms. It integrates **multi-factor authentication (MFA)**, **kernel-level security**, and **real-time system monitoring** into a single interactive dashboard.

This project highlights the concept of **defense in depth**, where multiple layers of security work together to protect the system from unauthorized access and cyber threats.

---

## 🚀 Features

### 🔐 Authentication System
- Username & Password login
- OTP-based multi-factor authentication
- Biometric authentication (simulated)
- Liveness detection for biometric verification
- Session token generation

### 📊 Dashboard
- Real-time CPU & Memory usage simulation
- Authentication events tracking
- Failed login attempts monitoring
- Interactive charts (CPU trends, auth trends)

### ⚙️ Kernel Operations
- User Mode vs Kernel Mode (Ring levels)
- System call simulation
- Kernel module loading/unloading
- Rootkit (malicious module) detection
- Interrupt handling & context switching

### 🔍 Process Monitoring
- View active processes with details:
  - PID, CPU usage, Memory usage
  - Threads, Priority, State
- Create new processes dynamically

### 📋 Event Logging
- Logs all system activities with timestamps
- Tracks authentication, kernel, and process events
- Categorized logs (info, success, error)
